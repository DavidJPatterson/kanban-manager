## Short description (132 chars max)
Multi-pod Kanban dashboard with flow metrics, cycle time analytics, and health charts for Azure DevOps.

## Detailed description

Kanban Manager gives engineering teams a live, multi-pod Kanban dashboard that pulls data directly from Azure DevOps.

FEATURES

- Multi-pod overview — track multiple teams from a single dashboard
- Full Kanban board with drag-and-drop column transitions
- Arrival rate and throughput charts with week-on-week trends
- Cycle time scatter plots (In Progress to Closed and Arrival to Closed)
- Throughput per person, items closed/resolved per person with sparklines
- WIP trend, age distribution, flow efficiency, stale item detection
- Bug ratio trend and throughput predictability (coefficient of variation)
- Burndown by Target PI with ideal-line overlay
- Filters by assignee, type, aged items, and free-text search
- Auto-refresh in the background (configurable interval)
- Quick-glance popup with key stats

PRIVACY

Your Personal Access Token and all data stay on your machine. The extension only communicates with dev.azure.com — no analytics, no telemetry, no third-party services.

GETTING STARTED

1. Install the extension
2. Click the icon and go to Settings
3. Enter your Azure DevOps org, project, PAT, and add your pods/teams
4. Click Test Connection, then Save
